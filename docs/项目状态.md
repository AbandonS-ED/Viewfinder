## 6. 风险与应对

| 风险 | 触发条件 | 应对 |
|---|---|---|
| Riverpod 2.x Notifier API 跟旧 Provider 不同 | Provider 写错 | 锁定 `flutter_riverpod ^2.5.0`，照官方 Notifier 文档写 |
| 缩略图加载阻塞 UI | GridView 同步 await Image | 用 `StreamProvider` 异步加载 + shimmer 骨架 |
| ConnectionNotifier 测试难 | 要 fake transport | 注入 `CameraTransport` 接口 fake，不依赖 PtpipSession |
| shared_preferences 在测试中需要 mock | 测试运行时无存储 | 用 `SharedPreferences.setMockInitialValues({})` 标准 pattern |
| iOS Runner 缺失（Phase 0 只创建 Android） | 想跑 iOS 真机 | Phase 2 仍只 Android；iOS 创建挪到 Phase 3 |
| Android scoped storage 限制 RAW 入相册 | Phase 3 | 评估 `gal` 包；不支持则降级到 "写入 app 专属目录 + 提示用户" |
